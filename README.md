# Portafolio_2025

Analista de datos con base financiera. Proyectos de automatización, análisis de datos y reporting.

## Demo

[Ver portafolio en vivo (ES)](https://vikoo255.github.io/Protafolio_2025/) | [View portfolio live (EN)](https://vikoo255.github.io/Protafolio_2025/en/)

## Tecnologías utilizadas

- HTML5
- CSS3 (con CSS Grid, Flexbox y variables CSS para temas)
- JavaScript (para toggle de tema)
- Diseño responsivo
- Modo oscuro/claro
- Optimizado para GitHub Pages

## Estructura del proyecto

- `index.html`: Página principal del portafolio (español)
- `en/index.html`: Versión en inglés
- `styles.css`: Estilos CSS (compartidos)
- `README.md`: Este archivo

## Cómo usar

1. Clona este repositorio
2. Abre `index.html` en tu navegador
3. Para desplegar en GitHub Pages: ve a Settings > Pages > Source: main branch /root
